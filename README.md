## TeambookMM
Um projeto simples para inserir nomes, email e número de telefone de membros da equipe.

Usa Angular para o frontend e Node.js com Express para o backend, armazenando os dados em um arquivo JSON.

Utilizei uma versão diferente do NeDB que não possui problemas de deprecate com algumas funções do Angular mais recente.
Disponível em: https://www.npmjs.com/package/@seald-io/nedb

Rodar o comando abaixo para instalar a dependência do NeDB:
npm install @seald-io/nedb

Rodar o comando abaixo na pasta backend para iniciar o mesmo:
npm start

Desenvolvido e executado no Visual Studio 2026, em localhost com Chrome.
